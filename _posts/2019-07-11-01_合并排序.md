## 算法思想
合并排序是分治法当中一个经典的排序算法，用分治策略实现对n个元素进行排序，用合并算法将n/2组元素进行两两排序，再将n/4组元素进行两两排序，直至整个元素排好顺序。

## 时间复杂度
时间复杂度 o(nlogn)

## java代码
```java
public class mergeSort {

    public static void main(String[] args) {
        int p[] = new int[]{85, 3, 52, 9, 7, 1, 5, 4, 12};
        MergeSort(p);
        for (int a : p) {
            System.out.print(a + " ");
        }
    }

    /**
     * 拆分数组区间，不断分支
     */
    public static void MergeSort(int p[]) {
        int length = p.length;
        if (length > 1) {
            int middle = length / 2;
            int pA[] = Arrays.copyOfRange(p, 0, middle);// 拆分问题规模
            int pB[] = Arrays.copyOfRange(p, middle, length);
            MergeSort(pA);
            MergeSort(pB);
            Merge(pA, pB, p);
        }
    }

    public static void Merge(int pA[], int pB[], int p[]) {
        int i = 0;
        int j = 0;
        int k = 0;
        while (i < pA.length && j < pB.length) {
            if (pA[i] <= pB[j]) {
                p[k] = pA[i];
                i++;
            } else {
                p[k] = pB[j];
                j++;
            }
            k++;
        }
        if (i == pA.length) {
            while (k < p.length) {
                p[k] = pB[j];
                k++;
                j++;
            }
        } else if (j == pB.length) {
            while (k < p.length) {
                p[k] = pA[i];
                k++;
                i++;
            }
        }
    }
}
```
